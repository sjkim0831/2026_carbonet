# SI/SM 어시스턴트 프로그램 카탈로그

## 목적
- SI(구축)와 SM(운영/유지보수) 업무에 붙일 수 있는 어시스턴트 프로그램 유형을 넓게 정리한 목록이다.
- Carbonet 같은 업무 시스템에 단계적으로 붙일 수 있도록 `무엇을 도와주는지` 중심으로 분류했다.
- 실제 구현 시에는 단일 대형 에이전트보다 `작은 전용 어시스턴트 여러 개`로 나누는 편이 안전하다.

## 1. 요구사항/기획 지원
1. 요구사항 정리 어시스턴트
   - 회의 내용, 메신저, 메일을 정리해 기능 요구사항으로 변환
2. 업무 프로세스 분석 어시스턴트
   - 현행 업무 흐름을 단계별로 문서화
3. 메뉴/화면 정의서 생성 어시스턴트
   - 메뉴명, URL, 접근권한, 화면 목적을 자동 정리
4. 기능 명세 초안 생성 어시스턴트
   - 입력값, 출력값, 예외 처리, 권한 조건을 문서화
5. 변경 영향도 분석 어시스턴트
   - 특정 기능 수정 시 영향 받는 화면/테이블/API를 추적
6. 일정/공수 산정 어시스턴트
   - 요구사항 규모를 기준으로 개발/테스트/배포 공수 추정
7. 제안요청서(RFP) 분석 어시스턴트
   - 발주 문서를 구조화하고 개발 항목으로 분해
8. 산출물 체크리스트 어시스턴트
   - 프로젝트 단계별 필수 문서 누락 여부 확인

## 2. 화면/메뉴/권한 메타데이터 지원
1. 페이지 등록 점검 어시스턴트
   - 화면 템플릿, URL, 컨트롤러 연결 상태 확인
2. 메뉴 자동 등록 어시스턴트
   - 메뉴 테이블과 다국어 메뉴명 등록
3. 프로그램 기능 등록 어시스턴트
   - 프로그램 ID, 기능명, 실행 URL, 메뉴 매핑 생성
4. 공통코드 등록 어시스턴트
   - 그룹코드/상세코드 존재 여부 확인 및 자동 생성
5. 권한 매핑 어시스턴트
   - 역할별 메뉴/프로그램 접근권한 등록
6. 버튼 권한 점검 어시스턴트
   - 조회/등록/수정/삭제/승인 버튼 권한 누락 확인
7. 다국어 리소스 등록 어시스턴트
   - 한/영 화면명, 메시지 키, 라벨 생성
8. 화면-메뉴-권한 일관성 검증 어시스턴트
   - 실제 URL과 메뉴/권한 메타데이터 불일치 탐지
9. 미사용 메뉴 정리 어시스턴트
   - 더 이상 호출되지 않는 메뉴/프로그램 식별
10. 메뉴 트리 재구성 어시스턴트
   - 상위/하위 메뉴 구조를 자동 정렬 및 검증

## 3. 개발 지원
1. 화면 스캐폴딩 어시스턴트
   - Controller, Service, Mapper, Template 기본 코드 생성
2. CRUD 생성 어시스턴트
   - 목록/상세/등록/수정/삭제 기본 기능 생성
3. API 명세 생성 어시스턴트
   - 요청/응답 DTO와 엔드포인트 초안 생성
4. DB Mapper 생성 어시스턴트
   - MyBatis XML, Mapper 인터페이스, VO/DTO 생성
5. SQL 초안 생성 어시스턴트
   - 조회 조건과 조인 구조에 맞는 SQL 작성
6. 배치 작업 생성 어시스턴트
   - 정기 실행 작업, 이력 적재, 실패 재처리 초안 작성
7. 파일 업로드 처리 어시스턴트
   - 업로드 정책, 저장 경로, 검증 로직 생성
8. 엑셀 업로드/다운로드 어시스턴트
   - 템플릿 생성, 컬럼 매핑, 오류 리포트 생성
9. 알림 연동 어시스턴트
   - 메일, SMS, 알림톡, 내부 알림 발송 로직 초안 작성
10. 외부 연계 인터페이스 어시스턴트
   - 타 시스템 API 연동 구조 및 예외 처리 구성
11. 로그 코드 삽입 어시스턴트
   - 업무 식별자 중심으로 추적 로그 추가
12. 예외 처리 표준화 어시스턴트
   - 공통 에러 응답과 예외 클래스 구조 정리
13. 레거시 코드 해석 어시스턴트
   - 오래된 서비스/쿼리/컨트롤러 동작 설명
14. 코드 리팩터링 어시스턴트
   - 중복 제거, 계층 분리, 네이밍 정리 제안
15. 성능 개선 포인트 탐지 어시스턴트
   - N+1, 비효율 조인, 과도한 조회 범위 분석

## 4. 테스트/품질 지원
1. 테스트 케이스 생성 어시스턴트
   - 정상/오류/경계 조건 시나리오 자동 작성
2. 회귀 테스트 후보 추출 어시스턴트
   - 수정 범위 기준으로 테스트 대상 기능 선정
3. QA 버그 재현 어시스턴트
   - 사용자 보고 내용을 재현 가능한 절차로 변환
4. 입력값 검증 점검 어시스턴트
   - 필수값, 길이, 형식, 중복 검증 누락 탐지
5. 화면 검수 체크리스트 어시스턴트
   - 레이아웃, 문구, 버튼, 메시지, 권한 표시 확인
6. API 계약 검증 어시스턴트
   - 문서와 실제 응답 구조 차이 검출
7. 배포 전 점검 어시스턴트
   - 설정값, 마이그레이션, 권한, 코드 누락 점검
8. 장애 재발 방지 어시스턴트
   - 장애 원인 기반으로 테스트 항목 추천

## 5. 운영/SM 지원
1. 장애 접수 분류 어시스턴트
   - 문의를 장애/개선/단순문의/데이터수정으로 분류
2. 운영 문의 답변 초안 어시스턴트
   - 사용자 문의에 대한 운영자 응답 초안 생성
3. 장애 원인 추적 어시스턴트
   - 로그, 배포 이력, DB 상태를 기반으로 후보 제시
4. 운영 이슈 트리아지 어시스턴트
   - 긴급도와 영향도를 기준으로 우선순위 지정
5. 재현 환경 구성 어시스턴트
   - 운영 이슈를 개발/검증 환경에서 재현하는 절차 제시
6. 운영 데이터 수정 스크립트 어시스턴트
   - 안전한 UPDATE/INSERT 스크립트 초안 작성
7. 배포 영향 안내 어시스턴트
   - 사용자 공지, 운영자 안내문 자동 작성
8. 정기 점검 체크 어시스턴트
   - 배치, 인터페이스, 인증서, 용량, 오류 로그 확인
9. SLA 대응 어시스턴트
   - 장애 응답시간, 처리시간, 에스컬레이션 관리
10. 유지보수 이력 정리 어시스턴트
   - 티켓, 커밋, 배포 이력을 기능 단위로 묶어 정리
11. 운영 매뉴얼 생성 어시스턴트
   - 자주 하는 문의/처리 절차를 문서화
12. 야간 장애 대응 어시스턴트
   - 즉시 확인할 로그/프로세스/DB 체크포인트 안내

## 6. 데이터/DB 지원
1. ERD 해석 어시스턴트
   - 테이블 관계와 업무 의미 설명
2. 테이블 영향도 분석 어시스턴트
   - 특정 컬럼 변경 시 관련 화면/API 탐지
3. 데이터 품질 점검 어시스턴트
   - NULL, 중복, 참조 무결성 위반 후보 탐지
4. 기준정보 관리 어시스턴트
   - 공통코드, 기준코드, 분류체계 등록/정비 지원
5. 마이그레이션 스크립트 생성 어시스턴트
   - DDL/DML 변경문 초안 작성
6. 인덱스 추천 어시스턴트
   - 느린 쿼리 기준으로 인덱스 후보 제안
7. 데이터 정합성 비교 어시스턴트
   - 운영/검증/이관 대상 데이터 차이 비교
8. 개인정보 마스킹 어시스턴트
   - 개발용 샘플 데이터 비식별 처리
9. 데이터 백필 어시스턴트
   - 신규 컬럼/코드 체계 도입 시 기존 데이터 보정
10. 배치 데이터 검증 어시스턴트
   - 적재 건수, 누락, 중복, 실패 이력 점검

7. CI 실패 분석 어시스턴트
   - 빌드/테스트 실패 원인 후보 제시
8. 배포 승인 자료 생성 어시스턴트

7. 운영 데이터 수정 검토 어시스턴트
   - 공통코드/권한/메뉴/기준정보 보정 스크립트 검증

10. 유지보수 티켓 처리 어시스턴트
   - 티켓을 분석해 코드 수정과 메타데이터 수정을 분리

## 13. 추천 운영 방식
- `읽기 전용 분석형`과 `자동 반영 실행형`을 분리한다.
- `자동 반영 실행형`은 승인 대상 범위를 명확히 둔다.
- 메뉴/권한/공통코드 자동 생성은 반드시 감사 로그를 남긴다.
- 운영계는 즉시 반영보다 `검토 후 승인` 모델이 안전하다.
- 처음에는 전부 자동화하지 말고 `추천 -> 검토 -> 반영` 순서로 도입한다.
