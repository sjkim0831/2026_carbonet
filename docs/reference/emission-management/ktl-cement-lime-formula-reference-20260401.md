# KTL 시멘트/석회 변수 정리 반영 기준

원본 파일: `KTL_시멘트_석회_변수 정리.hwpx`

정리 목적:
- HWPX 본문과 이미지 표를 사람이 읽을 수 있는 한글 텍스트 기준으로 고정
- 시멘트/석회 계산식을 코드와 SQL 시드에 동일하게 반영할 때 기준점으로 사용

## 추출 범위

- 본문 섹션 텍스트 4개
- 이미지 4개를 PNG로 변환 후 육안 판독
- 표 2.1, 표 2.4 수기 전사

## 시멘트 생산

### 티어 선택 플로우 이미지 요약

- 특정부서 혹은 국가 수준의 시멘트 생산 자료가 없으면 Tier 4 배출 없음
- 특정부서의 이산화탄소 배출에 대한 상세 활동자료가 있으면 Tier 3
- 국가 클링커 생산 자료가 가능하면 Tier 2
- 아니면 핵심부문 여부 판단 후 Tier 1 또는 더 높은 Tier 자료 수집

### 변수 정의

- `Mci`: 생산된 I 유형 시멘트의 무게(질량), 톤
- `Ccli`: I 유형 시멘트의 클링커 비율, 비율
- `Im`: 클링커 소비를 위한 수입, 톤
- `Ex`: 클링커의 수출, 톤
- `Mcl`: 생산된 클링커의 무게(질량), 톤
- `Md`: 킬른에 재활용되지 않은 CKD의 무게(질량), 톤
- `Cd`: CKD에서 원래 탄산염의 비율, 비율
- `Fd`: CKD에서 원래 탄산염의 소성 비율, 비율
- `Mi`: 킬른에서 소비된 탄산염 `i`의 무게(질량), 톤
- `Fi`: 탄산염 `i`에 대해 달성된 소성 비율, 비율
- `Mk`: 유기물 또는 기타 탄소를 포함하는 비연료 원료 `k`의 무게(질량), 톤
- `Xk`: 비연료 원료 `k`에서 총 유기물 또는 기타 탄소의 비율, 비율

### 계산식

- Tier 1: `[SUM(Mci × Ccli) - Im + Ex] × EFclc`
- Tier 2: `Mcl × EFcl × CFckd`
- Tier 3: `SUM(EFi × Mi × Fi) - Md × Cd × (1 - Fd) × EFd + SUM(Mk × Xk × EFk)`

### 비고

- `EFclc = 0.52`
- `EFc = 0.4397`
- `EFcl = 0.510`
- `CFckd = 1 + (Md / Mcl) × Cd × Fd × (EFc / EFcl)`
- `CFckd = 1.020` if 자료 없음
- `EFd = 0.4397`
- 케로겐 또는 기타 탄소의 열량 기여가 총 열량의 5% 미만이면 `(Mk × Xk × EFk) = 0`

## 석회 생산

### 티어 선택 플로우 이미지 요약

- 탄산염 투입량과 화학 성분 및 직접 배출계수 자료가 있으면 Tier 3
- 생산 석회 유형별 생산량 자료가 있으면 Tier 2
- 아니면 유형별 석회 생산량에 대한 국가 기본 배출계수로 Tier 1 적용

### 변수 정의

- `Ml,i`: `i` 유형의 석회 생산, 톤
- `MgO`: MgO 함유량
- `CaO 함유량`: 톤 CaO/톤 석회
- `Mi`: 소비된 탄산염 `i`의 무게(질량), 톤
- `Fi`: 탄산염 `i`에 대해 달성된 소성 비율, 비율
- `Md`: LKD의 무게(질량), 톤
- `Cd`: LKD 내 원래 탄산염의 무게 비율, 비율
- `Fd`: LKD에 대해 달성된 소성 비율, 비율

### 계산식

- Tier 1: `SUM(EF석회,i × Ml,i)`
- Tier 2: `SUM(EF석회,i × Ml,i)`
- Tier 3: `SUM(EFi × Mi × Fi) - Md × Cd × (1 - Fd) × EFd`

### 비고

- 고칼슘석회 기본 배출계수: `0.75`
- 고토석회 기본 배출계수: `0.86` 또는 `0.77`
- 수경성석회 기본 배출계수: `0.59`
- 고토석회는 선진국에 높은 값, 개도국에 낮은 값 적용
- Tier 2는 LKD 보정이 없다고 가정
- 석회 유형 값이 없으면 `85% 고칼슘 + 15% 고토석회 + 0% 수화석` 가정으로 `0.75`

### Tier 2 배출계수 식

- 고칼슘석회: `EF석회,a = SRCaO × CaO 함유량`
- 고토석회: `EF석회,b = SRCaO·MgO × CaO·MgO 함유량`
- 수경성석회: `EF석회,c = SRCaO × CaO 함유량`

## 코드 반영 원칙

- 시멘트 Tier 1은 `- Im + Ex` 부호를 그대로 유지
- 시멘트 Tier 2는 `Mcl × EFcl × CFckd` 구조를 그대로 유지
- 시멘트/석회 Tier 3의 `EFi`, `EFd` 는 표 2.1 기반 탄산염 종류 매핑을 우선 사용
- 석회 Tier 1과 Tier 2는 유형별 입력을 우선 사용하고, 문서의 default 값은 fallback으로만 사용
