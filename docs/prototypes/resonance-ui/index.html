Onboarding

Project Proposal Matrix

Matrix

제안서 원본, 생성 결과, 현재 런타임, 패치 타깃을 프로젝트 단위 매트릭스로 비교합니다.

Proposal Mapping Draft

AI Draft

Scenario

Project Design Output

Design

Release Unit

Performance Stack Governance

Speed

Log & Scheduler Governance

Ops

Module Pattern Governance

Module

AI로 추가하는 설치형 모듈도 같은 깊이와 같은 구조로 생성되는지 패턴 가족 기준으로 검토합니다.

Module Intake Queue

AI Review

새 모듈 폴더와 외부 모듈 후보를 AI가 분석한 뒤 ownership, CSS, DB, rollback 관점으로 검토합니다.

Module Intake Analysis

Analysis

AI가 정규화한 모듈 후보의 공통/프로젝트 경계, DB 영향, CSS 충돌, provenance를 검토합니다.

Module Attach Plan

Attach

모듈이 release unit과 runtime package에 어떤 형태로 포함되는지 승인 전 미리 확인합니다.

Module Selection Popup

Select

화면 생성 중 필요한 설치형 모듈을 체크하거나 영향도 팝업을 열어 선택합니다.

Module Selection Checklist

Check

선택한 모듈들이 dependency, CSS, DB, rollback 조건을 모두 만족하는지 확인합니다.

Module Selection Result

Applied

선택한 모듈이 실제로 어떤 페이지, 백엔드, DB, 패키지 자산에 반영됐는지 한 번에 확인합니다.

Module Selection Trace

Trace

모듈 선택 결과가 generation run, release unit, repair 세션과 어떻게 연결되는지 확인합니다.

Chain & Matrix Explorer

Trace

메뉴, 시나리오, 페이지, 컴포넌트, API, 백엔드, DB, release unit, runtime target을 한 화면에서 추적합니다.

Module Binding Preview

Preview

선택한 모듈이 프로젝트, 시나리오, release unit, runtime package에 어떻게 묶이는지 확인합니다.

Style Dedupe Governance

CSS

모듈 CSS 중복, 누락, 테마/토큰 재정의 여부를 점검해 스타일 드리프트를 차단합니다.

Missing Asset Queue

Gap Closure

빠진 페이지, 컴포넌트, 바인딩, 요구사항 갭을 큐로 관리하고 repair 세션으로 연결합니다.

Repair Workbench

Patch

선택 화면과 요소를 기준으로 현재 런타임과 비교하고 재사용 중심으로 수정한 뒤 패치 릴리즈를 만듭니다.

Backend Chain Explorer

Trace

route, controller, service, mapper, SQL, DB object까지 체인으로 추적하고 보안/권한 바인딩도 같이 봅니다.

Current Runtime Compare

Parity
